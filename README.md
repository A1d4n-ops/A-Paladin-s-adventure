
# Paladin Adventure

Explore the world as a paladin who starts from the bottom, and slowly grow stronger as you invoke the power of god onto your foes

## Installation

Python 3.10.11 is needed to play this game. [Python 3.10.11]([https://www.python.org/downloads/](https://www.python.org/downloads/release/python-31011/)) to install the correct version of Python.
Thonny version 4.1.7 is required to play this game as well.[Thonny 4.1.7](https://thonny.org/) to install Thonny.

## Description

This game involves the player making decisions where depending on what is chosen, it can change your adventure drastically.
The player must choose their adventure as some routes might lead to certain doom
With a variety of decisions, this also means that there are many branches from each decision, some even looping back to past decisions.

## Run the Program

```python
python game.py
```
